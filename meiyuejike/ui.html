解答：

UI设计怎么才能学好呢？

1、学习 UI 得先把基础的软件运用好，例如:Adobe 的 Photoshop,AI 等。这是做好设计的前提。此外最好能有点美术功底。

2、软件熟练使用后，就要开始看设计类的有关书籍，因为 UI 包括三个方面，视觉设计，交互设计与用户体验。等到软件熟练掌握后，就要开始看一些设计方面的书籍，比如如何编排？如何配色？如何做界面？另外交互的书籍、用户体验书籍必须要看的，只有这三种结合才是真正的 UI 设计师

3、等看完书籍后，相信你对 UI 应该有更好的认识。当你有时间的时候你可以去一些好的网站去看下一些前辈们的文章，这样有利于你少走弯路。

4、书籍、文章每天可抽出时间看一些。另外的学习，可以通过视频教程来加强自己薄弱的方面。

学习 UI 设计到底需不需要美术基础？

其实，简单来说，绘画对于 UI 设计是一种表达手段，是最基础的一个职业技能，绘画只占 UI 设计工作很小的一部分，绝不是主要的职业需求。作为一个 UI 设计从业人员，可以将绘画用作辅助的手段，而不必以其他行业的高标准来要求自己的绘画能力。

UI设计师的未来职业发展方向有哪些呢？

1.智能硬件

智能硬件是各种 APP 必经发展之路，很多 APP 现在并非刚需，产品渠道和需求越发越模糊和不明显

2.VR 虚拟现实三维 UI 设计

我们在 07 年前刮起三维设计学习风，因为当时中国硬件设备条件，现在发展只有少数几家规模大公司应用影视业，现在因为有了 VR 设备如智能眼镜等，这里的设计只依托于引擎的计算，对硬件设备要求相对要求较低，开发成本低，UI 设计在 VR 上发展市场有相当大空间，设计思维会更广阔。

3.用户体验，做好交互与产品

很多 UI 设计师，现在一直停留在 GUI 阶段，而大公司对人才的需求也很专一，导致设计师发展有所局限，但是传统企业需要认识和互联网企业在中小公司看来，给设计师参与用户研究和产品的考察对设计师来说很有帮助，帮助企业产品功能打开局限，争取更多用户与利益。动图

4.设计技能与职位晋级

企业与企业文化不一，讲究企业对产品和视觉的重要性也不一，UI 设计师始终在界面上玩转，细节因为工作流程等原因越发模糊，UI 设计师必须要意识到我们交互细节，加强对自身设计细节的理解，对用户在视觉上的追求根据设计行业趋势不断打磨自己，像视觉设计师和设计总监看齐。对产品在视觉上做好硬功。

5. 设计传道者

中国的设计行业需求很多，而在早期的美工平面设计等阶段还有相当一部分人才需要技能过度，你担当其中的传道者也是一个选择，根据自身条件考虑。设计师本身就是一个自我思维与视野不断打破的过程，我们要时刻创新自己，根据行业和趋势随时调整自己的技能和方向。
